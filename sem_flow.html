sem_tomo.flow

sem_tomo.flow.gen_xfalign_comand(fp_in: FilePath) → None

hardcoded xfalign -pa -1 -pr source.mrc align.xf

sem_tomo.flow.gen_align_xg(fp_in: FilePath) → None

hardcoded xftoxg -ro -mi 2 {WORKDIR}/align.xf {WORKDIR}/align.xg

sem_tomo.flow.convert_tif_to_mrc(file_path: FilePath) → int

generates source.mrc assumes there’s tifs in input dir uses all the tifs in dir # tif2mrc {DATAPATH}/*.tif {WORKDIR}/Source.mrc

sem_tomo.flow.create_stretch_file(tilt: float, fp_in: FilePath) → None

creates stretch.xf used to gen corrected.mrc file looks like: 1 0 0 {TILT_PARAMETER} 0 0

where TILT_PARAMETER is calculated as 1/cos({TILT_ANGLE}). Note that tilt angle is input in degrees, however cos method expects radians

sem_tomo.flow.gen_newstack_mid_mrc_command(fp_in: FilePath) → None

generates mid.mrc newstack -secs {MIDZ}-{MIDZ} {WORKDIR}/{BASENAME}.mrc {WORKDIR}/mid.mrc

sem_tomo.flow.gen_keyimg(fp_in: FilePath) → Dict

generates keyimg (large thumb) mrc2tif -j -C 0,255 mid.mrc {WORKDIR}/keyimg_{BASENAME}.jpg

sem_tomo.flow.gen_keyimg_small(fp_in: FilePath) → Dict

convert -size 300x300 {WORKDIR}/hedwig/keyimg_{BASENAME}.jpg -resize 300x300 -sharpen 2 -quality 70 {WORKDIR}/hedwig/keyimg_{BASENAME}_s.jpg
