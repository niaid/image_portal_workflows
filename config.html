config.command_loc(cmd: str) → str

Given the name of a program that is assumed to be in the current path, return the full path by using the shutil.which() operation. It is assumed that shutil is available and the command is on the path :param cmd: str, the command to be run, often part of the IMOD package :return: str, the full path to the program
